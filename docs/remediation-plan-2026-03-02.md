# 보완 계획 및 구현 완료 내역 (2026-03-02)

## 계획
1. 자동 종료 보장
- 게임 시작 시 서버 타이머를 등록하고, 만료 시 게임 종료 및 종료 이벤트를 브로드캐스트한다.

2. 입력 유효성 강화
- 셀 클릭 좌표(row, col) 범위를 서버에서 검증해 비정상 입력을 차단한다.

3. 규칙 상수 일관화
- freeze 해제 이벤트 대기 시간을 하드코딩 대신 `DEFAULT_CONFIG.freezeDuration`으로 통일한다.

4. 회귀 테스트 보강
- 자동 종료/좌표 검증 케이스를 테스트에 추가한다.

## 구현 결과
- `GameManager`에 게임 종료 리스너와 자동 종료 타이머 로직 추가.
- `clickCell`에 보드 경계 체크 추가 (`Invalid cell position` 반환).
- 소켓 핸들러에서 게임 종료 리스너 등록 후 `GAME_ENDED` 자동 송출.
- freeze 해제 타이머를 설정 상수 기반으로 변경.
- 테스트 2건 추가:
  - 범위 외 좌표 클릭 실패
  - 시간 만료 자동 종료 + 종료 콜백 호출
